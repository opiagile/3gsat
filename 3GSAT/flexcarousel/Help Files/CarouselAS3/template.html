flashSQL.componentInstance

Availability

Flash Player 7.
FlashSQL Version 2.0

Usage

flashsqlInstance.componentInstance = "instanceName:String"; 

Parameters

flashsqlInstance: Instance name of the FlashSQL on the stage
instanceName: Instance name of the component to load data into. Please note, this is not the component type.

Returns

Nothing.

Description

This property can be set using the component inspector panel or at runtime. It sets the instance name of the particular component you intend to load data into once a select query has been executed. Please note that this will not work if you have not set the component type as well.

Example

myFlashSQL.componentInstance = "myComboBox";
