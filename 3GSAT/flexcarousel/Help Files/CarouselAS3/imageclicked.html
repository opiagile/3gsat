imageClicked

event imageClicked [Event]

Availability

Actionscript 2.0, 3.0; Flash Player 7, 8, 9+.

Usage

event imageClicked; //An Event

Parameters

carouselInstance: Instance name of the carousel component on the stage.

Returns

An Event

Description

This is an event that the component broadcasts when the an image is clicked and gains focus as the current image. When using the AS2 version, an object is passed that contains information about the clicked image. The data passed are:

1. index: The index of the image

2. src: The source of the image (url)

3. description: The description attached to the image via the xml file.

Please see the sample below to know how to listen and use these properties.

Example

For Actionscript 3:
carouselInstance.addEventListener("imageClicked", imageClickedListener);
function imageClickedListener(evt:Event):void{
	trace("Image Clicked");

}


For Actionscript 2:
var imageClickedListener:Object = new Object();
imageClickedListener.imageClicked = function(dat:Object){
	trace(dat.description);
}
