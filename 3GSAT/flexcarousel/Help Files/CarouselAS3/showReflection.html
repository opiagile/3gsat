carousel.showReflection

public showReflection : Boolean [read/write]

Availability

Actionscript 3.0; Flash Player 9+. //Please note that this feature is only available for Actionscript 3.0 versions.

Usage

carouselInstance.showReflection = value; //Default Value is true

Parameters

carouselInstance: Instance name of the carousel component on the stage.
value: A Boolean value to assign to the property.

Returns

Nothing.

Description

This property can be set using the component inspector panel or in the actionscript code. This property determines whether to show the reflection of each of the images after bein loaded. Setting it to true means the reflections are shown and false means otherwise.

Example

carouselInstance.showReflection = true;
