carousel.getTotalImages()

public getTotalImages() : Array [read Only]

Availability

Actionscript 2.0, 3.0; Flash Player 7, 8, 9+.

Usage

carouselInstance.getTotalImages; //Returns an array of the loaded images

Parameters

carouselInstance: Instance name of the carousel component on the stage.

Returns

An Array of the images loaded. The gallery array contains object instances of each image with src and title_txt properties.

Description

This method returns an array of all the loaded images. It returns an array of obejects with the following properties:

1. src: This is the source url of the image.

2. title_txt: This is the title associated with the image instance as defined in the xml file.

Example

for(var i:int=0; i<carouselInstance.getTotalImages.length; i++){
	
	trace(carouselInstance.getTotalImages[i].src);


}
