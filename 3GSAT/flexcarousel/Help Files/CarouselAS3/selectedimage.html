carousel.selectedImage()

public selectedImage() : Object [read Only]

Availability

Actionscript 2.0, 3.0; Flash Player 7, 8, 9+.

Usage

carouselInstance.selectedImage; //Returns an object data containing the currentlty selected image

Parameters

carouselInstance: Instance name of the carousel component on the stage.

Returns

An Object data of the currently selected image.

Description

This method returns an object of the currently selected image. It returns an array of obejects with the following properties:

1. src: This is the source url of the image.

2. description: This is the title associated with the image instance as defined in the xml file.

3. index: This is the index of the image.

Example

trace(carouselInstance.selectedImage.src);
