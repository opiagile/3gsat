carousel.fadeImages

public fadeImages : Boolean [read/write]

Availability

Actionscript 2.0, 3.0; Flash Player 7, 8, 9+.

Usage

carouselInstance.fadeImages = value; //Default Value is false

Parameters

carouselInstance: Instance name of the carousel component on the stage.
value: A Boolean value to assign to the property.

Returns

Nothing.

Description

This property can be set using the component inspector panel or in the actionscript code. This property determines whether to fade the images as they appear distant from the center to the back of the pack. This implies that the current movie will always have 0 transparency. The default value is false.

Example

carouselInstance.fadeImages = true;
