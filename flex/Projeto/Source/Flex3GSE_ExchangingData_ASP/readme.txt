Flex 3 Getting Started Experience

This is an importable Flex Builder project ZIP file. To use the files in this
file, you must import the ZIP file into Flex Builder.

To import this file:
1) Open Flex Builder.
2) Select File > Import > Flex Project from the menu bar.
3) Select the Archive file radio button.
4) Click Browser, navigate to the ZIP file, and select Open.
5) Click Finish. Flex Builder imports the project. 

Once imported as a project, you can view, modify, and run the MXML files it contains.

Note: Some projects contain additional assets, such as databases, and related 
ColdFusion, PHP, JSP, or ASP files. You must copy these files to the appropriate 
location before you can use them locally.
